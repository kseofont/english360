# 📍 How to Find Test Setup - Step by Step Guide

## 🎯 Three Ways to Access Test Setup

### Method 1: Quick Setup Script (FASTEST - No menu needed!)

1. **Make sure you're logged in as Administrator**
2. **Copy this URL and paste in your browser:**
   ```
   yoursite.com/wp-content/plugins/tutor-scheduling-booking/test-quick-setup.php
   ```
3. **Replace `yoursite.com` with your actual site URL**
4. **Press Enter** - The script will create everything automatically!

**✅ This is the EASIEST method - no menu navigation needed!**

---

### Method 2: WordPress Admin Menu

#### Step-by-Step Navigation:

1. **Login to WordPress Admin**
   - Go to: `yoursite.com/wp-admin`
   - Enter your admin username and password
   - Click "Log In"

2. **Find the "Tutor" Menu**
   - Look at the **LEFT SIDEBAR** of the WordPress admin
   - Scroll down until you see **"Tutor"** (this is the Tutor LMS plugin menu)
   - It usually appears after "Pages" and before "Comments"

3. **Click on "Tutor"**
   - Click the word **"Tutor"** to expand the menu
   - You'll see submenu items appear below it

4. **Find "Scheduling Test Setup"**
   - In the expanded Tutor submenu, look for:
     - Courses
     - Students  
     - Instructors
     - Settings
     - Scheduling
     - **Scheduling Test Setup** ← **THIS ONE!**

5. **Click "Scheduling Test Setup"**
   - This will open the test setup page

### Method 3: Direct URL (If menu doesn't show)

**Copy and paste this URL in your browser:**

```
yoursite.com/wp-admin/admin.php?page=tutor-scheduling-test
```

**Replace `yoursite.com` with your actual site URL**

**Example:**
- If your site is `localhost/lms`, use: `localhost/lms/wp-admin/admin.php?page=tutor-scheduling-test`
- If your site is `mysite.com`, use: `mysite.com/wp-admin/admin.php?page=tutor-scheduling-test`

---

### Method 4: Via Scheduling Main Page

1. **Go to: Tutor > Scheduling** (in WordPress admin menu)
2. **At the top of that page**, you'll see a blue info box
3. **Click the button "📋 Go to Test Setup Page"** in that box
4. This will take you directly to the test setup

## 🔍 Still Can't Find It?

### Troubleshooting Steps:

**1. Check if plugin is activated:**
   - Go to: **Plugins > Installed Plugins**
   - Look for "Tutor Scheduling & Booking"
   - Make sure it says **"Activated"** (not "Deactivate")
   - If not activated, click "Activate"

**2. Check user permissions:**
   - You must be logged in as **Administrator**
   - Regular users won't see this menu
   - Go to: **Users > Your Profile** to check your role

**3. Clear browser cache:**
   - Press `Ctrl + F5` (Windows) or `Cmd + Shift + R` (Mac)
   - Or clear browser cache completely

**4. Try direct URL method:**
   - Use Method 3 above (Direct URL)
   - This bypasses the menu completely

**5. Use Quick Setup Script:**
   - Use Method 1 above
   - This is the easiest and doesn't require any menu navigation
